Attack Flow

1
SE Console
SE picks scenario, enters target IP, clicks Launch
2
CALDERA Operation
POST /caldera/api/v2/operations (via nginx proxy)
3
Sandcat Agent
Agent on victim VM receives abilities from CALDERA
4
Techniques Execute
PowerShell commands run on victim (MITRE ATT&CK mapped)
5
Sophos Detects
Endpoint/NDR/Firewall generates real-time alerts
6
SE Pivots to XDR
Show detections in Sophos Central -- the proof point

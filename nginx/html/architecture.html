• AI Provider — Anthropic Claude, OpenAI, Google Gemini, or local Ollama

Scenario Briefings — select your scenario, understand the narrative, know your proof point
SCN-001
Credential Dumping Chain
CISO Technical
Attacker breaches an endpoint, injects into a privileged process to dump LSASS credentials from memory, then uses harvested hashes to access additional systems without knowing the password.
~3 min T1003 · T1055 · T1078
★ Proof Point
Sophos XDR alert → "LSASS Memory Read" — show the process tree, parent-child chain, and severity score. Ask: "Would your SOC catch this at 2am?"
SCN-002
Discovery & Enumeration Chain
Technical All
Attacker quietly maps the environment — system info, running processes, local files, and network interfaces — building a target picture before escalating. Classic post-exploitation recon.
~2 min T1082 · T1083 · T1057 · T1016
★ Proof Point
Sophos XDR alert → "Suspicious System Enumeration" — show the correlated timeline across four techniques. Great for demonstrating XDR correlation vs. siloed alerts.
SCN-003
Phishing → Persistence
Executive All
Simulates the most common initial access path: a phishing payload establishes a foothold, then immediately sets up persistence via registry run keys and a scheduled task to survive reboots.
~3 min T1566 · T1547 · T1053
★ Proof Point
Sophos XDR alert → "Registry Run Key Modification" + "Scheduled Task Creation" — show both alerts firing within seconds. Perfect for the "they were already in for 30 days" conversation.
SCN-005
Pass-the-Hash Lateral Movement
CISO Technical
Attacker uses a stolen NTLM hash — never cracking the actual password — to authenticate as a privileged user and move laterally via SMB to another system on the network.
~3 min T1550 · T1021
★ Proof Point
Sophos XDR alert → "Pass-the-Hash Authentication" — show the source and destination hosts in the alert. The question: "How many of your systems are reachable this way today?"
SCN-006
PowerShell AMSI Bypass
Technical
Attacker patches the Windows AMSI (Anti-Malware Scan Interface) in memory before executing malicious PowerShell, blinding traditional AV scanning while the payload runs in plain sight.
~2 min T1562 · T1059
★ Proof Point
Sophos XDR alert → "AMSI Bypass Detected" — show that Sophos detected the bypass attempt despite the attacker's evasion. Demonstrates behavioral detection succeeding where traditional AV scanning is blind.
SCN-008
Ransomware Simulation
Executive Board-Level
Full ransomware kill chain: disable security tools, delete Volume Shadow Copies to destroy backups, encrypt files, and stage a ransom note. The highest-impact scenario — resonates with executive and board-level audiences.
~4 min T1486 · T1490
★ Proof Point
Sophos XDR alert → "Shadow Copy Deletion" — every CISO recognizes this as the point of no return. Show the alert firing before encryption starts and ask: "How fast do you need to respond?"
Universal Demo Execution — same 6 steps for every scenario
1
Open SE Console → select your scenario from the catalog → enter the victim VM's IP address → click LAUNCH ATTACK
2
Attack Console opens automatically — left panel shows the live CALDERA technique feed (abilities + status), right panel loads the victim Windows desktop via RDP
3
Narrate each technique as it executes — watch the left panel feed and walk the prospect through what the attacker is doing and why. The right panel shows the actual Windows desktop responding in real time.
4
Pivot to Sophos XDR — switch to Sophos XDR and show real-time detections mapping to MITRE ATT&CK. This is the proof point — see the alert card above for what to highlight for your specific scenario.
5
Return to SE Console → scroll to the Enrich panel → click to generate an AI narrative. The configured AI provider produces a customer-facing incident summary tailored to the scenario — include in prospect follow-up or debrief documentation.
6
Close with the question — "This is exactly what your SOC team would see in real time. How long would it take your team to identify, triage, and contain this today?"
